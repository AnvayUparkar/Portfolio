# Project Images

This directory contains images for the project detail modal.

## Expected Images

Place your project images here. The modal expects the following images:

- `health-ai-1.png` & `health-ai-2.png` - AI-Driven Diagnostics project images
- `jarvis-1.png` - JARVIS project image
- `quickcourt-1.png` & `quickcourt-2.png` - QuickCourt project images
- `faculty-ratio-1.png` - Student-Faculty Ratio project image
- `ngo-tracker-1.png` - NGO Event Tracker project image

## Image Specifications

- **Format**: PNG (recommended) or JPG/WebP
- **Aspect Ratio**: Preferably 4:5 (mobile portrait) or similar
- **Optimization**: Optimize images for web to reduce file size
- **Resolution**: 800x1000px minimum (will be scaled as needed)

If no image is provided for a project, a placeholder with the project number will be shown instead.
